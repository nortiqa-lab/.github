# Nortiqa Lab

**Diseñamos, validamos y escalamos agentes de IA auditables para industrias reales.**

Minería · Energía · Agro · Construcción · Sector Público

---

## Qué hacemos

Nortiqa Lab es una fábrica de agentes de IA, no una agencia. Cada agente que construimos es verificable, reversible y desplegable en la infraestructura del cliente.

| Vertical | Casos de uso |
|----------|-------------|
| ⛏️ Minería | Análisis operacional, alertas tempranas, procesamiento de sensores |
| 🛢️ Energía & Oil | Monitoreo de operaciones, gestión documental automatizada |
| 🌾 Agro | Reportes automáticos, asistentes de decisión para productores |
| 🏗️ Construcción | Seguimiento de obra, control de avance, alertas de desvío |
| 🏛️ Sector Público | Procesamiento de expedientes, consulta normativa, atención ciudadana |

---

## Stack

```
FastAPI · Docker · n8n · Ollama / Claude · PostgreSQL · Redis · Caddy
```

Infraestructura propia en Hetzner (Patagonia). Sin dependencia de nubes de terceros para los datos del cliente.
